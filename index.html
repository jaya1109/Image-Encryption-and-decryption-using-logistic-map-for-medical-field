🔐 Image Encryption & Decryption

Using Logistic Map | Medical Image Security

📌 Project Overview

This project focuses on securing medical images using a logistic map-based chaotic encryption technique. The logistic map generates highly random sequences that are used to encrypt image pixels. The encrypted image becomes unreadable and can only be restored using the correct key, ensuring strong security and protection of sensitive medical data.

⚙️ Working Process

  • Input image
  • Grayscale conversion
  • Logistic sequence generation
  • Image encryption
  • Image decryption

📌 Conclusion

This project demonstrates the application of chaos theory in image encryption. The logistic map provides high randomness and sensitivity, making the encryption secure and reliable. The system ensures that the image is transformed into an unreadable format and can be accurately recovered. This method is highly suitable for securing sensitive medical images.
