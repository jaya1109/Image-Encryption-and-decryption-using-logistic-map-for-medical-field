🔐 Image Encryption & Decryption

Using Logistic Map | Medical Image Security

📌 Project Overview

This project focuses on securing sensitive medical images using a logistic map-based chaotic encryption technique. In modern healthcare systems, protecting patient data is extremely important during transmission and storage. Traditional encryption techniques may not provide sufficient randomness, so this project uses chaos theory to enhance security. The logistic map generates highly unpredictable sequences which are used to encrypt image pixels. The image becomes unreadable after encryption and can only be restored using the correct key. This ensures strong data protection and prevents unauthorized access.

🎯 Objectives

  • Secure medical image transmission
  • Prevent unauthorized access
  • Use chaos theory for encryption
  • Ensure data confidentiality

🧠 Concept

The logistic map equation:

x(n+1) = r × x(n) × (1 − x(n))

This generates chaotic sequences that are highly random and sensitive. These sequences are used to encrypt image data.

⚙️ Working Process

  • Input image is taken
  • Converted to grayscale
  • Logistic sequence is generated
  • Image is encrypted
  • Decryption restores original image
