🔐 Image Encryption & Decryption

Using Logistic Map | Medical Image Security

📌 Project Overview

This project secures images using logistic map-based chaotic encryption. The logistic map generates pseudo-random sequences that encrypt pixel values, making the image unreadable without the correct key.

🧠 Concept

Logistic map is a chaotic function used for randomness: x(n+1) = r × x(n) × (1 − x(n)) It is highly sensitive and used for secure encryption systems.

⚙️ Working

  • Input Image
  • Convert to Grayscale
  • Generate Chaotic Sequence
  • Encrypt Image
  • Decrypt using Key
