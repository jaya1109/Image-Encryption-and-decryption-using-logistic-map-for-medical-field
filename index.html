🔐 Image Encryption & Decryption

Using Logistic Map | Medical Image Security

📌 Project Overview

This project focuses on securing medical images using logistic map-based chaotic encryption. The system converts images into an unreadable encrypted format and restores them using a secret key.

🎯 Objectives

  • Secure medical image transmission
  • Apply chaos theory in encryption
  • Prevent unauthorized access

🧠 Concept

The logistic map generates pseudo-random chaotic sequences using the formula: x(n+1) = r × x(n) × (1 − x(n)). These sequences are used to encrypt image pixels.

⚙️ Working Process

  • Input Image
  • Convert to Grayscale
  • Generate Logistic Sequence
  • Encrypt Image
  • Decrypt Image
