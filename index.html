🔐 Image Encryption & Decryption

Using Logistic Map | Medical Image Security

📌 Project Overview

This project secures medical images using logistic map-based chaotic encryption. It converts images into an unreadable format and restores them using a key.

🧠 Concept

Logistic map generates chaotic random sequences: x(n+1) = r × x(n) × (1 − x(n))

⚙️ Working Process

  • Input Image
  • Convert to Grayscale
  • Generate Logistic Sequence
  • Encrypt Image
  • Decrypt Image
