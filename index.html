🔐 Image Encryption & Decryption

Using Logistic Map | Medical Image Security

📌 Project Overview

This project focuses on securing medical images using a logistic map-based encryption technique. The logistic map is a chaotic mathematical function that generates highly random sequences. These sequences are used to encrypt image pixels, making the image unreadable to unauthorized users. The same key is used to decrypt the image and restore it to its original form.

🎯 Objectives

  • Ensure secure transmission of medical images
  • Prevent unauthorized access
  • Apply chaos theory in encryption
  • Maintain data confidentiality and integrity

🧠 Concept

The logistic map is defined as:

x(n+1) = r × x(n) × (1 − x(n))

This equation generates chaotic sequences that are highly sensitive to initial conditions. These sequences are used to modify pixel values of the image, ensuring strong encryption.

⚙️ Working Process

  • Input image is taken
  • Image is converted to grayscale
  • Logistic map generates chaotic sequence
  • Pixel values are encrypted
  • Encrypted image is produced
  • Using same key, image is decrypted
