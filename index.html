🔐 Image Encryption & Decryption

Using Logistic Map | Medical Image Security

Developed by Jayasudha

📌 Project Overview

This project focuses on securing sensitive medical images using a logistic map-based chaotic encryption technique. In modern healthcare systems, protecting patient data is extremely important during transmission and storage. Traditional encryption techniques may not provide sufficient randomness, so this project uses chaos theory to enhance security. The logistic map generates highly unpredictable sequences which are used to encrypt image pixels. The image becomes unreadable after encryption and can only be restored using the correct key, ensuring strong data protection and preventing unauthorized access.

🎯 Objectives

  • To design a secure image encryption system using logistic map-based chaotic sequences
  • To protect sensitive medical images from unauthorized access during transmission
  • To implement highly random encryption using chaos theory
  • To ensure accurate decryption using the same key
  • To maintain data confidentiality and integrity

🧠 Concept

This project is based on chaos theory using the logistic map equation:

x(n+1) = r × x(n) × (1 − x(n))

The generated sequence is highly sensitive and unpredictable. This chaotic behavior is used to shuffle and modify image pixels, converting the original image into an encrypted unreadable format.

⚙️ Working Process

  • Input medical image is taken
  • Converted into grayscale
  • Logistic map generates chaotic sequence
  • Pixels are shuffled and encrypted
  • Encrypted image is generated
  • Same key is used for decryption
  • Original image is restored

📌 Conclusion

This project demonstrates the effective use of chaos theory in securing digital images using logistic map-based encryption. The system transforms the original image into a highly random encrypted format, ensuring strong data protection. Due to the sensitivity of the logistic map to initial conditions, the encryption becomes highly secure and difficult to break. The decryption process accurately reconstructs the original image using the same key, ensuring reliability. Compared to traditional encryption methods, this approach provides enhanced randomness and better resistance to attacks. It is highly suitable for securing sensitive medical images where privacy and data integrity are essential. Overall, this project presents a practical and efficient solution for secure image transmission in modern digital systems.
